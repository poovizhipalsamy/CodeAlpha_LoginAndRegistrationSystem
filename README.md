# CodeAlpha_LoginAndRegistrationSystem
## Features

* User Registration
* User Login
* Username and Password Validation
* File Handling
* Stores User Credentials

## Technologies Used

* C++
* File Handling (fstream)
* Strings
* Functions

## How It Works

1. Register a new account using a username and password.
2. User details are stored in a file.
3. Login using registered credentials.
4. The system verifies the username and password.
5. Displays success or error messages.

## Learning Outcomes

* File Handling
* Functions
* String Manipulation
* User Authentication
